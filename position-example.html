This div is Fixed, positioned in the bottom right. Here's how this one was positioned:
.fixed {
position: fixed;
bottom: 0;
right: 0;
}
This div is static.
There is no code needed, because
static is the default, but to make it explicit:
.static {
position: static;
}





This div is sticky, set to stay at the top when it gets there. Here's the how this one was set up:
.sticky {
position:sticky;
top: 0;
}










This is an absolute div positioned to the right. Here's how this one was done: .absolute {
position:absolute;
right:100px;
}









This is a relative div, positioned a little to the left of the last div. Here's how it was done: .relative {
position:relative;
left: 50px;
}






This is a div reset to an initial value. Here's how it got that way:
.initial {
position: initial;
}
